# Portfolio Image Assets — Gemini Generation Guide

## Style Baseline for All Illustrations

Flat vector kawaii chibi style. Soft pastel palette with deep navy (#1A1A2E) and sky blue (#4FC3F7) accents.
Transparent or white background. No text in image. Studio Ghibli-inspired warmth. Clean bold outlines.
No emojis, no flags, no text.
